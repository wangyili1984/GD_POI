# AMap POI Data Miner for Urban Analysis 
### (高德地图POI数据挖掘工具 - 城市研究版)

## 📌 项目背景 (Project Background)
[cite_start]本项目是我在城市中心区街区尺度研究及实践中开发的效率工具。

在探索城市中心区街区尺度研究的过程中，我发现获取高质量的城市语义数据（POI）是获取城市街区质量反馈的基础。本项目旨在通过“傻瓜式”的操作逻辑，帮助规划师和研究者快速获取城市功能要素，为后续的强化学习仿真提供环境输入。

## 🌟 核心功能 (Core Features)
- **零门槛操作**：针对非计算机专业背景的规划师设计，只需配置高德 API Key 即可运行。
- **多维度爬取**：支持按城市、按矩形区域、按分类关键字获取 POI 数据。
- **数据自动化清洗**：自动剔除冗余字段，输出符合城市分析需求的 CSV 格式数据。
- **地理坐标转换**：支持火星坐标系（GCJ-02）处理，便于导入 GIS 或 BIM 系统。

## 🛠️ 快速开始 (Quick Start)
1. **获取 API Key**：前往高德开放平台申请 Web 服务 Key。
2. **配置文件**：在 `config.py`（或相关脚本位置）中填入您的 Key。
3. **运行脚本**：执行 `main.py` 并根据提示输入目标区域。
4. **获取结果**：数据将自动保存在 `/output` 文件夹下。
